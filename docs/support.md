# LWA Support

If you need help with LWA, use the official LWA support channel provided on your website, Whop product, or onboarding materials.

## Common Issues

### Backend connection problems

- Confirm the backend URL is reachable
- Check the in-app API base URL in Settings
- Retry with a public video URL

### Clip generation failures

- Try a different public video URL
- Wait a moment and retry if the service is under load
- Confirm the source video is publicly accessible

### Billing and access questions

- Manage your plan through the official LWA web storefront
- Contact support if your paid access is not reflected correctly

## What To Include In A Support Request

- The video URL you submitted
- The approximate time the issue occurred
- A screenshot of the error message
- Your device model and iOS version
