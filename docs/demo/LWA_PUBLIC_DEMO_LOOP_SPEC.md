# LWA Public Demo Loop Spec

## Purpose

This route explains LWA to a first-time visitor in under a minute.

The demo loop is intentionally finite:

`landing -> first mission -> source added -> clips ready -> recovery available -> proof saved -> Signal Sprint -> marketplace teaser -> return loop`

## What the user should understand

- LWA starts from one source.
- LWA ranks clip opportunities by retention behavior.
- LWA can return rendered output when available.
- LWA can fall back to strategy-only when rendering is unavailable.
- Lee-Wuh is the guide, not decoration.
- Missions and rewards are local/demo first.
- Signal Sprint is the first playable game layer.

## What is live now

- Public demo loop UI
- Local stage progression
- Lee-Wuh guidance copy
- Signal Sprint decision prompt
- Readiness checklist

## What is intentionally disabled

- real payouts
- crypto
- hidden mining
- direct social posting automation
- backend mutations from demo interactions
- marketplace claims that imply real revenue
