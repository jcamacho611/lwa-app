# LWA Operations Playbook

## Audience

Maria / operations team.

## Purpose

Give operations clear plain-English tasks to help LWA make money, organize launch, support sales, prepare Whop, assist investor outreach, and coordinate with Justin.

## What LWA is

LWA is an AI clipping and content repurposing product.

The product starts with a simple promise:

> Drop in a creator source and get ranked short-form clip packages with hooks, captions, timestamps, scores, and posting angles.

## What LWA can honestly say now

Use only after the latest deployed app is smoke-tested:

- LWA is a live AI clipping MVP.
- LWA produces ranked clip packages.
- LWA supports hooks, captions, timestamps, scores, and packaging copy.
- Upload-first workflows are supported where enabled.
- Public URLs are best-effort because platforms can block extraction.
- Whop can be used as an external sales/access path.

## What LWA cannot promise yet

Do not say:

- guaranteed viral
- guaranteed income
- works with every platform URL
- direct posting is live
- marketplace payouts are live
- blockchain/NFT features are live
- Whop paid access is verified until webhook deployment is configured and tested

## Daily responsibilities

1. Check the live frontend.
2. Check the Whop purchase/access page.
3. Collect demo leads.
4. Track customer questions.
5. Send follow-up messages.
6. Record bugs or confusing moments.
7. Send Justin a daily update.

## Whop setup checklist

- Confirm Whop product page is live.
- Confirm pricing/offer copy is accurate.
- Confirm Whop product points to the Railway frontend.
- Confirm support/contact info is visible.
- After webhook merge/deploy, confirm Whop test webhook succeeds.

## Sales support checklist

For each lead, collect:

- name
- niche/business type
- source type they create from
- main platform
- pain point
- whether they need done-for-you clipping
- whether they want demo or checkout

## Demo booking script

Use:

> Hey, we built LWA — an AI clipping MVP that turns creator sources into ranked short-form clip packages with hooks, captions, timestamps, scores, and posting angles. We’re onboarding early users now. Want a quick walkthrough?

## Customer follow-up script

Use:

> Thanks for checking out LWA. The best way to test it is to bring one real source you already want clipped. We’ll show you what the system finds, which clip it ranks first, and what hooks/captions it recommends.

## Daily update to Justin

Send:

- leads contacted
- demos booked
- buyers sent to Whop
- top objections
- bugs found
- customer quote or insight
- what you need unblocked

- book demos every week
- collect proof clips
- document customer wins carefully
- avoid guaranteed-income claims
- prepare investor update
- keep Whop page aligned with live product truth

## Legal/safety language

Always include:

> Public URL support is best-effort. Earnings and content performance vary. No guaranteed virality or income.
