# LWA Lee-Wuh World Background Asset

## Purpose

This asset is the separated Lee-Wuh world/background layer. It should be used as the environment layer behind the living Lee-Wuh character and optional weapon layers.

## What it contains

- Black/gold/purple mythic sci-fi environment
- Floating platform
- Portal/pillars
- Mist, crystals, and energy particles

## What it must not contain

- No Lee-Wuh character
- No sword
- No UI overlay

## Repo placement

```text
brand-source/lee-wuh/separated-assets/lee-wuh-world-background.glb
lwa-web/public/brand/lee-wuh/backgrounds/lee-wuh-world-background.glb
scripts/blender/create_lwa_lee_wuh_world_background_blend.py
```

## Usage rule

Use the `brand-source` copy as the canonical source asset. Use the `lwa-web/public` copy only for frontend loading from the browser.
