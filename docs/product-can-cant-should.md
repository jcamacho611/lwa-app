# What LWA / IWA Can Do

- Accept supported source inputs currently wired in the app.
- Generate ranked clip candidates.
- Provide hooks, captions, timestamps, scores, and post order.
- Provide why-this-matters reasoning, thumbnail text, and CTA suggestions.
- Return rendered asset links when processing succeeds.
- Return strategy packages when rendering or provider conditions are limited.
- Surface plan, credit, and feature metadata from the backend response.
- Preserve saved local run history in iOS when available.

# What LWA / IWA Cannot Do Yet

- Guarantee views, virality, payouts, or revenue.
- Fully replace a human editor in every workflow.
- Direct-post to TikTok, Instagram, YouTube, or Twitch.
- Browse Whop campaigns natively unless that integration is explicitly implemented.
- Submit campaign links automatically unless that workflow is explicitly implemented.
- Guarantee private or login-gated source ingestion.
- Guarantee AI-provider success when keys or upstream services are unavailable.
- Act as a full account, workspace, or team system yet.

# What LWA / IWA Should Do

- Help users move faster from source content to short-form clip packages.
- Make the strongest clip obvious.
- Keep fallback states useful instead of dead-ending the workflow.
- Keep pricing and upgrade prompts logical after value is shown.
- Keep platform, Whop, and App Store claims honest.
- Support future campaign workflows without pretending they are fully complete now.
