# LWA Brand Assets

## Brand Identity

### LWA (Technical/Product)
- Used for: app, company, repository, backend, frontend, technical references
- Focus: Professional video clipping and generation tools
- Style: Clean, functional, creator-native

### LEE-WUH (Mascot/Brand)
- Used for: mascot, merch, cultural brand moments, hero character
- Character: "The Last Creator" - cute but dangerous anime final-boss chibi
- Style: African + Japanese fusion with American streetwear twist
- Palette: Gold / Purple / Black luxury colors
- Features: Big jeweled dreads, overlord/creator-god aura, furry creature energy

## Asset Placement

### Primary Mascot Asset
- `lee-wuh-mascot.png` - Main mascot character
- Usage: Homepage hero, loading states, empty states, mobile splash, Whop cover, social posts

### Brand Applications
- Homepage hero side character
- Loading/processing state mascot

- Whop cover image

- "Lee-Wuh says..." creator tips

## Brand Rules

1. **LEE-WUH** for mascot/merch/cultural brand moments
2. **LWA** for app/company/backend/frontend/technical references
3. Maintain premium mythic feel without cluttering product flow
4. Use mascot at edges/brand moments, not blocking core functionality
